# Sandhya Gupta - Professional Portfolio

A modern, responsive portfolio website showcasing professional experience, skills, and projects.

## Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Modern UI**: Clean and professional design with smooth animations
- **Easy Navigation**: Smooth scrolling and intuitive menu
- **Professional Sections**:
  - Hero section with introduction
  - About section with achievements and education
  - Experience timeline
  - Projects showcase
  - Skills and technical expertise
  - Contact information

## How to Use

1. Open `index.html` in any modern web browser
2. Navigate through sections using the top menu
3. All contact links are functional (email, phone, LinkedIn)

## Customization

To customize the portfolio:

1. **Colors**: Edit CSS variables in `styles.css` (lines 9-17)
2. **Content**: Update text in `index.html`
3. **Images**: Add your photo by replacing the hero section
4. **Projects**: Add or modify project cards in the projects section

## Technologies Used

- HTML5
- CSS3 (with CSS Grid and Flexbox)
- JavaScript (Vanilla)
- Font Awesome Icons

## Browser Support
